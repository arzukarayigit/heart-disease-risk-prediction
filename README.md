# Heart Disease Risk Prediction

This project aims to predict heart disease risk using machine learning techniques and clinical health data.

## Project Overview

In this project:

- Missing data analysis and preprocessing were performed.
- Exploratory Data Analysis (EDA) was conducted.
- Correlation analysis and visualizations were created.
- Logistic Regression model was developed.
- Model performance was evaluated using:
  - Accuracy
  - Confusion Matrix
  - Classification Report

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Jupyter Notebook

## Model Performance

The Logistic Regression model achieved approximately **83% accuracy** on the test dataset.

## Dataset

The project uses the Heart Disease UCI dataset containing clinical and demographic patient information.
